# 方法组合策略

不同学习方法之间并非独立，而是存在协同效应。以下基于认知科学原则和实证研究，给出经验证的组合策略。

## 组合原则

1. **理解先于巩固**：先用费曼/精细追问确保理解，再用主动回忆/间隔重复巩固记忆
2. **局部先于全局**：先用组块化/双重编码掌握局部单元，再用交错练习/深度研究建立全局连接
3. **强度递增**：先单一方法建立基础，再组合增加认知挑战
4. **输出驱动**：所有方法最终指向"能输出"——能写、能讲、能做

## 核心组合详解

### 组合一：间隔重复 + 主动回忆（记忆巩固黄金搭档）

最可靠的长期记忆巩固组合。主动回忆是每次复习时的操作方式（怎么复习），间隔重复决定复习的时间节点（什么时候复习）。

**实施**：
1. 用主动回忆进行每次复习（闭书提取而非重读）
2. 每次提取后评分，输入SM-2算法
3. 算法决定下次复习间隔
4. 失败的知识点回到费曼/精细追问重新理解

### 组合二：费曼技巧 + 精细追问（深度理解组合）

费曼暴露盲区，精细追问深挖盲区的根源。这是"从说不清楚到能说清楚"的最短路径。

**实施**：
1. 费曼讲解一个概念给虚拟听众
2. 标记卡壳处
3. 对每个卡壳处用精细追问：为什么这里卡住了？缺了什么前置知识？
4. 回补知识，再次费曼讲解

### 组合三：交错练习 + 主动回忆（考试备战组合）

交错练习训练方法选择和迁移能力，主动回忆确保每道题是真正的独立提取。

**实施**：
1. 混合至少3种题型的题库
2. 随机顺序，每题先分类再闭卷解答
3. 错题归因：是方法选错了还是选对了但执行错了？
4. 方法选错→回到深层原理（费曼）；执行错→回到组块化训练

### 组合四：组块化 + 双重编码（知识结构化组合）

组块化将离散信息压缩为结构化单元，双重编码为每个组块创建视觉线索。适合需要快速调用的复杂知识体系。

**实施**：
1. 识别知识间的内在关系
2. 用一张整合图/概念图表示整个组块
3. 为组块设计"触发词+图像"的双重编码线索
4. 通过闪卡训练"触发词→整个组块"的快速提取

## 场景→组合快速推荐

| 场景 | 主组合 | 补充方法 |
|------|--------|---------|
| 零基础入门新领域 | 引文网络→深度研究→费曼 | 精细追问 |
| 短期备考（<1周） | 主动回忆+交错练习 | 专注-发散（防疲劳） |
| 长期备考（>3月） | 间隔重复+主动回忆 | 交错练习+组块化 |
| 攻克长期薄弱点 | 费曼+精细追问 | 多AI交叉质询 |
| 学习复杂系统（如生物化学） | 组块化+双重编码 | 费曼+间隔重复 |
| 准备演讲/教学 | 费曼+多AI交叉质询 | 精细追问 |
| 编程技能提升 | 组块化+交错练习 | 主动回忆 |
| 语言学习 | 间隔重复+主动回忆 | 双重编码+交错练习 |
