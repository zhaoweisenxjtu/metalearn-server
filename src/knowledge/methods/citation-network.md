# 引文网络提纯（Citation Network Refinement）

## 原理

学术领域的信息爆炸使入门者无法判断哪些文献值得读。引文网络提供了一个天然的过滤机制：被整个领域反复引用的文献就是该领域的骨架。从奠基人的经典文献出发，沿着引用关系向上追溯演变、向下追踪影响，可以最高效地构建领域知识地图。

**核心机制**：PageRank 式信息提纯——引文网络本质上是一个有向图，高被引文献相当于高 PageRank 节点。类似 Google 搜索的排名逻辑，应用于知识获取。

## 适用场景

- 系统进入一个全新的学术领域
- 做文献综述或学位论文开题
- 判断一篇论文在领域中的位置和重要性
- 快速了解一个领域的发展脉络和核心争议

## 局限性

- 引文偏差：经典文献可能过时，新文献可能因网络效应未被充分引用
- 不适用于刚诞生的前沿领域（引文图太稀疏）
- 高被引≠正确——有些被广泛引用是因为被广泛反驳

## 实操步骤

1. **锁定奠基人**：问AI"该领域最具影响力的3-5位学者是谁？各自的代表作是什么？"
2. **下载核心文献**：获取上述代表作的全文
3. **提取二阶文献**：在核心文献的参考文献中，找出被引用≥3次的Top 5
4. **向上追溯**：找到核心文献引用的更早期里程碑文献
5. **向下追踪**：用Google Scholar查看哪些近期文献引用了这些核心文献

## 组合建议

- **+深度研究**：引文网络是深度研究的文献获取阶段
- **+精细追问**：每篇核心文献追问"这篇为什么重要？它解决了什么问题？"
- **+多AI交叉质询**：用引文网络中的不同立场文献互相质疑

## 对话示例

> **教练**："你想入门强化学习。先锁定奠基人：Sutton和Barto的《Reinforcement Learning: An Introduction》是这个领域的圣经。下载后看参考文献——哪些文献被他们反复引用？"
>
> **用户**："Bellman (1957) 的《Dynamic Programming》、Watkins (1989) 的 Q-Learning论文被频繁引用..."
>
> **教练**："对。这就是两条关键脉络：Bellman奠基数理基础，Watkins开创实际算法。现在顺着这两条线去Google Scholar看谁引用了它们——你会找到DQN、Policy Gradient这些后续发展。这样你就有了一个清晰的'文献导航图'。"
