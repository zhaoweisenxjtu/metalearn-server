# 循证方法库

以下方法均有认知科学/教育心理学实证支持，根据场景灵活选用。

| 方法 | 文件 | 核心原理 | 证据来源 |
|------|------|---------|---------|
| 间隔重复 | `spaced-repetition.md` | 递增间隔复习巩固长时记忆 | Cepeda et al. (2006) |
| 主动回忆 | `active-recall.md` | 闭书提取比重复阅读有效50%+ | Roediger & Karpicke (2006) |
| 费曼技巧 | `feynman.md` | 用简单语言解释暴露盲区 | — |
| 交错练习 | `interleaving.md` | 混合题型提升迁移能力 | Rohrer & Taylor (2007) |
| 精细追问 | `elaborative-interrogation.md` | 追问"为什么"建立深层连接 | — |
| 双重编码 | `dual-coding.md` | 文字+图像双通道加工 | — |
| 专注-发散切换 | `focus-diffuse.md` | 交替模式解决复杂问题 | Oakley《Learning How to Learn》 |
| 组块化 | `chunking.md` | 打包信息减少工作记忆负荷 | — |
| 引文网络提纯 | `citation-network.md` | 高频被引文献是领域骨架 | — |
| 多AI交叉质询 | `multi-ai-cross-exam.md` | 多视角发现盲区 | — |

## 场景快速选择

| 场景 | 推荐方法 | 组合建议 |
|------|---------|---------|
| 初次学习新概念 | 费曼技巧 + 精细追问 | 先费曼自检，再追问深挖 |
| 巩固长期记忆 | 间隔重复 + 主动回忆 | SM-2 排期后闭书提取 |
| 应对考试 | 交错练习 + 组块化 | 混合题型交叉练习 |
| 攻克难点 | 专注-发散切换 + 精细追问 | 专注 25min → 发散 10min → 追问 |
| 建立领域全景 | 引文网络 + 深度研究 | 先找奠基文献，再系统研究 |
| 验证理解 | 多AI交叉质询 + 费曼技巧 | 多视角审问后简单解释 |
| 记忆公式/流程 | 双重编码 + 组块化 | 文字+图示双通道编码 |
| 综合能力提升 | 见组合策略文件 | `strategies/combination-strategies.md` |
