# 标准化评估协议

提供可复用的评估对话模板和按层级分类的问题库，用于系统化诊断用户的理解程度。

## 评估对话模板

### 模板A：单概念快速评估（5分钟）

```
教练：我们来确认一下 [概念] 的掌握程度。三个问题：

Q1（L2 复现）：请合上书，默写 [核心公式/定义]。
Q2（L3 应用）：计算/解决这个标准问题：[例题]。
Q3（L4 迁移）：如果我把条件从 [A] 改成 [B]，答案会发生什么变化？为什么？

（根据回答判定层级和知识缺口）
```

### 模板B：多概念结构评估（15分钟）

```
教练：下面我会念8个概念名称，请把它们分类。不要按教材的章节分，
而是按照它们之间的深层关系分。分好后给每组起个名字。

概念列表：[概念1] [概念2] [概念3] [概念4] [概念5] [概念6] [概念7] [概念8]

（观察分类依据是表面特征还是深层原理）

追问：你分出了 [N] 组。现在想想，有没有一个概念可以同时属于两组？
如果有，说明什么？（检验知识网络的灵活性）
```

### 模板C：全面层级诊断（涵盖L1-L5）

```
现在对 [知识点] 做一次全面诊断：

L1 识别：下面哪个问题需要用到 [知识点]？（给4个不同知识点的典型问题）
L2 复现：合上书，把 [核心公式] 从头推导一遍。
L3 应用：在这个标准条件下，用 [公式] 计算出结果。
L4 迁移：现在条件变了：[修改条件]。还能用同样的方法吗？如果不能，需要怎么调整？
L5 元认知：给一个完全没学过的人讲这个。注意你讲的过程中哪个地方最容易让对方困惑。

（逐层评分，定位精确的知识缺口）
```

## 按层级的问题类型库

### L2 复现层问题

**目的**：检验能否独立写出核心内容

| 类型 | 示例 |
|------|------|
| 定义复述 | "闭书写出XX的定义。（不要求精确到课本原句，但必须覆盖所有关键要素）" |
| 公式默写 | "闭书写出XX公式，标注每个符号的含义。" |
| 推导复现 | "在纸上从XX定律出发，推导出YY公式。标注每一步用了什么定理。" |
| 要素列举 | "闭书列出构成XX的三个必要条件。" |

### L3 应用层问题

**目的**：检验在标准条件下能否正确应用

| 类型 | 示例 |
|------|------|
| 标准计算 | 教材课后题难度的直接应用题 |
| 模式识别 | "下面5道题中哪些可以用XX方法解决？" |
| 步骤解释 | "解这道题的每一步，解释你在做什么以及为什么这么做。" |
| 检查验证 | "这个计算结果合理吗？用两种不同的方法验证。" |

### L4 迁移层问题

**目的**：检验在变化条件下能否仍然正确应用

| 类型 | 示例 |
|------|------|
| 条件变式 | 改变题目中的条件参数（数值/边界/初始条件） |
| 跨章节综合 | 结合两个以上知识点的综合题 |
| 真实场景 | 将抽象问题嵌入实际场景（噪声数据/不完整信息） |
| 反向问题 | 已知结果，反推条件或方法选择 |
| 错误诊断 | "有人用XX方法做了这道题得到了错误的答案，他错在哪里？" |

### L5 元认知层问题

**目的**：检验方法论层面的理解和自我监控能力

| 类型 | 示例 |
|------|------|
| 方法选择 | "这道题有A、B、C三种解法，各自的优缺点？你选哪个，为什么？" |
| 陷阱识别 | "这道题哪里最容易出错？如果出题人想坑学生，他会在哪里设陷阱？" |
| 教学验证 | "教一个零基础的人学这个概念，列教学大纲和预计的难点。" |
| 知识连接 | "XX和YY两个看似不相关的概念，底层有什么共同原理？" |
| 边界探索 | "XX定理的三个条件，哪个最容易被误认为可以去掉？"
| 信心校准 | "做这道题，做完后先估计自己的正确率，然后对答案。" |

## 评估结果记录

每次评估后记录到 `~/.meta-learning/assessment-log.json`（格式见 `references/data-persistence.md`）：

```
评估日期: YYYY-MM-DD
评估主题: [主题]
评估类型: [单概念快速/多概念结构/全面层级诊断]
评估结果:
  L1: [通过/未通过]
  L2: [通过/未通过]
  L3: [通过/未通过]
  L4: [通过/未通过]
  L5: [通过/未通过]
知识缺口: [具体说明在哪层、哪个点上卡住]
下次评估建议: [XX天后，重点检测YY]
```

## 评估节奏建议

| 时间点 | 评估类型 | 目的 |
|--------|---------|------|
| 每天学习结束时 | 单概念快速评估（覆盖当天主题） | 确认当天学习效果 |
| 每周结束时 | 多概念结构评估 | 检验知识网络是否按深层原理组织 |
| 每阶段结束时 | 全面层级诊断 | 确认是否达成阶段目标层级 |
| 备考前2周 | 全面层级诊断（所有核心考点） | 查漏补缺，决定最后冲刺重点 |
